# Exercício 1.29 - Fundamentos da Linguagem

## Enunciado

Crie um programa que tenha um dicionário com os nomes, idades e cor dos olhos de 5 membros da sua família. Cada item do dicionário deve ter o nome da pessoa como chave e uma tupla associada contendo a idade e a cor dos olhos.

## Solução

```python
familia = {
    "Nome1": (idade1, "cor_olhos1"),
    "Nome2": (idade2, "cor_olhos2"),
    "Nome3": (idade3, "cor_olhos3"),
    "Nome4": (idade4, "cor_olhos4"),
    "Nome5": (idade5, "cor_olhos5"),
}

print("Informações da família:", familia)
```

## Explicação

Neste código, criamos um dicionário chamado `familia` contendo os nomes, idades e cores dos olhos de 5 membros da família. Cada item do dicionário tem o nome da pessoa como chave e uma tupla associada contendo a idade e a cor dos olhos. Em seguida, imprimimos o dicionário.
