# Exercício 7.12 - Arquivos, Módulos e Pacotes

## Enunciado

Documente o programa anterior usando *docstring* e crie um programa que utilize o `pydoc` para gerar a documentação do módulo criado no exercício anterior.

## Solução

```python
# No terminal
pydoc -w Utils
```

## Explicação

Neste exercício, o comando `pydoc -w Utils` é usado para gerar a documentação do módulo `Utils` criado no exercício anterior. A opção `-w` faz o `pydoc` criar um arquivo HTML com a documentação.
