# Exercício 1.28 - Fundamentos da Linguagem

## Enunciado

Crie um programa que tenha uma lista com os nomes de 5 membros da sua família.

## Solução

```python
familia = ["Nome1", "Nome2", "Nome3", "Nome4", "Nome5"]

print("Membros da família:", familia)
```

## Explicação

Neste código, criamos uma lista chamada `familia` contendo os nomes de 5 membros da família. Em seguida, imprimimos a lista.
