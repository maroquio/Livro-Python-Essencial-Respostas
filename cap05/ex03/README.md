# Exercício 5.3 - Funções

## Enunciado

Crie uma função que tenha 3 parâmetros: um número inteiro, um número real e um texto. A função deve imprimir os valores dos parâmetros na tela. Em seguida, chame a função passando os argumentos de forma posicional.

## Solução

```python
def imprime_valores(inteiro, real, texto):
    print(inteiro, real, texto)

imprime_valores(7, 3.14, "Olá")
```

## Explicação

A função `imprime_valores()` é definida com três parâmetros: `inteiro`, `real` e `texto`. Esta função imprime os valores desses três parâmetros. Chamamos a função `imprime_valores()` passando 7, 3.14 e "Olá" como argumentos.
