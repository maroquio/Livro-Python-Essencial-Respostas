# Exercício 7.15 - Arquivos, Módulos e Pacotes

## Enunciado

Utilize o comando `pip list` para listar os pacotes Python instalados no sistema.

## Solução

```python
# No terminal
pip list
```

## Explicação

Nesta solução, o comando `pip list` é usado para listar todos os pacotes Python instalados no sistema. Este comando exibe o nome de cada pacote junto com sua versão.
