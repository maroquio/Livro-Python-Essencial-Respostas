# Exercício 5.1 - Funções

## Enunciado

Crie uma função com dois parâmetros numéricos, que retorne a soma dos dois números. Em seguida, crie uma chamada para essa função passando argumentos para os parâmetros e mostrando o resultado na tela.

## Solução

```python
def soma(a, b):
    return a + b

resultado = soma(5, 10)
print(resultado)
```

## Explicação

O código acima define uma função `soma()` que aceita dois parâmetros, `a` e `b`. Esta função simplesmente retorna a soma desses dois parâmetros. Então chamamos a função `soma()` passando dois números como argumentos, 5 e 10, e armazenamos o resultado retornado em `resultado`. Finalmente, imprimimos o resultado.
