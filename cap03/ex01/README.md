# Exercício 3.1 - Estruturas de Repetição

## Enunciado

Crie um programa que mostre todos os números de 1 a 100.

## Solução

```python
for i in range(1, 101):
    print(i)
```

## Explicação

Neste programa, estamos usando o laço `for` para iterar sobre uma sequência de números. A função `range(1, 101)` gera uma sequência de números começando em 1 e terminando em 100 (o número 101 é exclusivo). Para cada número na sequência, a função `print()` imprime o número na saída padrão. A variável `i` é usada para armazenar o valor atual na sequência a cada iteração do laço.
