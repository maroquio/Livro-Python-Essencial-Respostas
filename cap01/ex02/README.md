# Exercício 1.2 - Fundamentos da Linguagem

## Enunciado

Crie um programa que armazene dois números em variáveis separadas e imprima a soma, subtração, multiplicação e divisão desses números.

## Solução

```python
num1 = 10
num2 = 5

soma = num1 + num2
subtracao = num1 - num2
multiplicacao = num1 * num2
divisao = num1 / num2

print(f"Soma: {soma}")
print(f"Subtração: {subtracao}")
print(f"Multiplicação: {multiplicacao}")
print(f"Divisão: {divisao}")
```

## Explicação

O código cria duas variáveis `num1` e `num2` que armazenam dois números. Em seguida, são criadas quatro variáveis: `soma`, `subtracao`, `multiplicacao` e `divisao` que armazenam o resultado das respectivas operações matemáticas. Por fim, o `print()` é utilizado para imprimir as quatro variáveis com as respectivas operações.
