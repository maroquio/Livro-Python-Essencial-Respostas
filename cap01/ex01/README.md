# Exercício 1.1 - Fundamentos da Linguagem

## Enunciado

Crie um programa que armazene seu nome e sua idade em variáveis separadas e imprima uma saída formatada com elas.

## Solução

```python
nome = "João"
idade = 30
print(f"Meu nome é {nome} e tenho {idade} anos.")
```

## Explicação

O código simplesmente cria duas variáveis `nome` e `idade` que armazenam o nome e a idade do usuário, respectivamente. Em seguida, o `print()` é utilizado para imprimir uma mensagem formatada com as variáveis armazenadas.
