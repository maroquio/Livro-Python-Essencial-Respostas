# Exercício 7.13 - Arquivos, Módulos e Pacotes

## Enunciado

Utilize o `pip` para instalar um pacote Python chamado `fastapi`.

## Solução

```python
# No terminal
pip install fastapi
```

## Explicação

Neste exercício, o comando `pip install fastapi` é usado para instalar o pacote `fastapi` no sistema. O `pip` é um sistema de gerenciamento de pacotes usado para instalar e gerenciar pacotes de software escritos em Python.
