# Exercício 7.14 - Arquivos, Módulos e Pacotes

## Enunciado

Utilize o `pip` para atualizar um pacote Python chamado `fastapi`.

## Solução

```python
# No terminal
pip install --upgrade fastapi
```

## Explicação

Neste exercício, para atualizar o pacote `fastapi`, foi usado o comando `pip install ---upgrade fastapi`. A opção `---upgrade` informa ao `pip` para atualizar o pacote se ele já estiver instalado.
