# Exercício 7.18 - Arquivos, Módulos e Pacotes

## Enunciado

Dentro do ambiente virtual "pratica\_venv", instale a biblioteca `requests` usando o comando `pip`.

## Solução

```python
# No terminal
pip install requests
```

## Explicação

Neste exercício, o comando `pip install requests` é usado para instalar a biblioteca `requests` dentro do ambiente virtual "pratica\_venv".
