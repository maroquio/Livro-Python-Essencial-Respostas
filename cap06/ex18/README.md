# Exercício 6.18 - Coleções

## Enunciado

Faça um programa que declare um valor inteiro, um valor *string* e um número real e, a partir deles, crie uma tupla com esses 3 valores, mostrando a tupla na tela ao final.

## Solução

```python
inteiro = 10
string = "olá"
real = 3.14

tupla = (inteiro, string, real)
print(tupla)
```

## Explicação

Nestas solução, temos três variáveis: um inteiro, uma *string* e um real. Criamos uma tupla com esses três valores usando parênteses. Ao fim, a tupla é impressa.
