# Soluções dos Exercícios - Python Essencial

Este repositório contém as soluções dos exercícios do livro **Python Essencial: Para Programadores** de Ricardo Maroquio.

## Estrutura

Cada exercício está em sua própria pasta contendo:
- **`capXX_exYY.py`** - Código Python da solução
- **`README.md`** - Enunciado, solução e explicação detalhada

## Como usar

1. Navegue até o capítulo desejado
2. Entre na pasta do exercício
3. Leia o `README.md` para ver enunciado e explicação
4. Execute o arquivo `.py` localmente

## Acesso Direto pelo Livro

No PDF do livro, você pode clicar no título de qualquer exercício para ser direcionado diretamente à solução correspondente neste repositório.
