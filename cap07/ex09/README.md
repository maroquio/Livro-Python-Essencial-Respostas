# Exercício 7.9 - Arquivos, Módulos e Pacotes

## Enunciado

Crie um programa que utilize o módulo nativo `random` para gerar um número aleatório entre 1 e 10.

## Solução

```python
import random
numero = random.randint(1, 10)
print(numero)
```

## Explicação

Aqui utilizamos o módulo `random` do Python, que fornece funções para gerar números aleatórios. A função `random.randint` gera um número aleatório dentro do intervalo fornecido, neste caso, entre 1 e 10. O resultado é então impresso na tela.
